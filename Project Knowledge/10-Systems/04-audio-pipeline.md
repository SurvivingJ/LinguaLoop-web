# Audio Pipeline

## Pipeline

1. Text (transcript) generated by ProseWriter agent
2. AudioSynthesizer agent processes:
   - Selects voice from `dim_languages.tts_voice_ids` (JSONB array)
   - Adjusts speed from `dim_languages.tts_speed`
   - Calls Azure Cognitive Services Speech SDK (`azure-cognitiveservices-speech`)
   - Generates MP3 audio data
3. Upload to Cloudflare R2 via `R2Service.upload_audio(slug.mp3, audio_data)`
4. Public URL constructed: `https://audio.linguadojo.com/{slug}.mp3`
5. `tests.audio_url` and `tests.audio_generated` updated in DB

## Azure TTS Configuration

- Requires: `SPEECH_KEY` and `SPEECH_REGION` env vars
- Uses SSML for voice control
- Neural voices per language
- Migrated from OpenAI TTS (see `AZURE_MIGRATION_GUIDE.md`)

## Storage

- **Cloudflare R2 bucket**: `linguadojoaudio`
- S3-compatible API via `boto3`
- Zero egress fees
- Public URL with custom domain
